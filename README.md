# PathPilot

PathPilot is a modern career guidance and exploration platform designed to help students discover suitable career paths, universities, degree programs, and future opportunities through an interactive and user-friendly experience.

The project aims to simplify the career planning process for students who are often confused about selecting the right field after matric, intermediate, or graduation. PathPilot provides organized career information, guidance resources, and smart recommendations in a clean and engaging interface.

## Features

* Career exploration and guidance.
* University and degree program information
* Interactive and modern UI/UX
* Smart recommendation-based experience
* Responsive design for desktop and mobile
* Smooth navigation and animations
* Beginner-friendly and easy-to-use interface

## Purpose of the Project

The main goal of PathPilot is to create a platform where students can easily explore their interests, understand different career options, and make better academic and professional decisions without confusion.

This project was developed as a learning and portfolio project while focusing on frontend development, UI/UX design, and interactive web experiences.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design
* Modern UI Components and Animations
